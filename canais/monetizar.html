1. AdsTerra - Banner

Importante: Cole o código COMPLETO, não apenas o ID.

Visualização do Banner

O banner aparecerá aqui quando você inserir um código válido

Como obter o código do AdsTerra:

  1. Crie uma conta em AdsTerra (link com referência)
  2. Após login, vá em "Inventory" → "Sites" e adicione seu site/TV
  3. Vá em "Ad Units" → "Banner" e crie um novo banner
  4. Escolha o tamanho (recomendado: 300x250, 728x90 ou 468x60)
  5. Copie o código completo fornecido
  6. Cole o código completo no campo acima

Dica: O banner aparecerá em tempo real na visualização acima.

2. AdsTerra - Intersticial

Cole o código completo, não apenas o ID.

O que é um anúncio intersticial?

É um anúncio de tela cheia que aparece em momentos específicos, como ao carregar o canal ou entre transições de conteúdo.

Como funcionará no seu canal:

O anúncio será exibido quando um espectador acessar seu canal pela primeira vez ou recarregar a página. Oferece maior visibilidade e geralmente tem CPM mais alto que banners.

Como configurar intersticiais:

  1. Crie uma conta em AdsTerra (link com referência)
  2. No painel, vá em "Ad Units" → "Interstitial"
  3. Clique em "Create Interstitial"
  4. Configure as opções (recomendado: carregar a cada 60 segundos)
  5. Copie o código completo fornecido
  6. Cole o código completo no campo acima

Nota: Intersticiais geralmente têm CPM mais alto que banners.

3. Doações (Apoia.se)

O link deve ser do Apoia.se (ex: https://apoia.se/seunome)

Como funciona o Apoia.se:

Plataforma brasileira para criadores receberem apoio financeiro. Taxa de apenas 6% sobre o valor arrecadado.

Pontual Meta Livre: Opção ideal para doações livres. Você recebe qualquer valor arrecadado (menos a taxa) sem precisar atingir uma meta mínima. Perfeito para doações pontuais para seu canal.

5. Sistema de Comerciais Pagos

92% para você, 8% para a plataforma

Como funciona:
• Espectadores podem inserir comerciais de até 90 segundos
• Preço: R$0,05 por segundo × quantidade de dias
• Pagamento via MercadoPago (Marketplace Seller)
• 92% para você, 8% para a plataforma
• Comerciais entram em fila e são exibidos em intervalos
