1. AdsTerra - Banner

Visualização do Banner

O banner aparecerá aqui quando você inserir o código

Como obter o código do AdsTerra:

  1. Crie uma conta em AdsTerra
  2. Vá em "Sites" → "Adicionar site" e informe a URL do seu canal
  3. Após aprovação do site, vá em "Ad Units" → "Banner"
  4. Clique em "Criar novo" e configure as dimensões do banner
  5. Selecione o seu site na lista de sites aprovados
  6. Clique no botão "Get Code" - uma janela se abrirá com o código JavaScript
  7. Copie todo o código JavaScript fornecido (incluindo as tags <script>)
  8. Cole o código completo no campo acima

2. AdsTerra - Intersticial

Anúncio Intersticial: É um anúncio de tela cheia que aparece ao carregar o canal. Tem maior visibilidade e ganhos mais altos que banners.

Como obter o código do Intersticial:

  1. Na sua conta do AdsTerra, vá em "Ad Units" → "Interstitial"
  2. Clique em "Criar novo" e escolha o formato intersticial
  3. Selecione o seu site aprovado na lista
  4. Configure as opções de exibição (frequência, etc.)
  5. Clique no botão "Get Code" - uma janela se abrirá com o código JavaScript
  6. Copie todo o código JavaScript fornecido
  7. Cole o código completo no campo acima

4. ExxoClick - Anúncios de Vídeo

Você precisa obter o código VAST tag para anúncios de vídeo no ExxoClick

Como configurar anúncios de vídeo no ExxoClick:

  1. Crie uma conta em ExxoClick
  2. No painel, vá em "Monetize" → "Video Ads"
  3. Clique em "Create Video Ad Unit"
  4. Selecione o tipo de anúncio de vídeo (pré-roll, mid-roll, post-roll)
  5. Configure as opções do player (tamanho, cor do botão skip, etc.)
  6. Após criar o ad unit, clique em "Get Code" para obter a VAST tag
  7. Copie todo o código VAST tag (XML) fornecido
  8. Cole o código completo no campo acima

Nota: O ExxoClick fornece anúncios de vídeo com botão "skip ad". Você receberá um código VAST tag (XML) que será integrado ao player de vídeo do seu canal.
